# 📥 YT-Downloader

A simple tool to download YouTube videos and audio directly from a link.

## ✨ Features

- Paste any YouTube link and download instantly
- Choose between **MP3** (audio only) or **MP4** (video)
- Select your preferred resolution for MP4 downloads:
  - 720p
  - 1080p
  - 1440p
  - 2160p (4K)

## 📋 Requirements

[ffmpeg](https://ffmpeg.org/download.html) must be installed and added to your Windows PATH.

1. Download ffmpeg from the [official website](https://ffmpeg.org/download.html)
2. Extract the archive and add the `bin` folder to your System Environment Variables (PATH)

## 🚀 Installation

### Option 1 — Executable (recommended)
Download the latest `.exe` from the [Releases](../../releases) page. No Python required.

### Option 2 — Run from source
1. Download or clone this repository
2. Open the project folder in [Visual Studio Code](https://code.visualstudio.com/)
3. Open the integrated terminal 
4. Run the script:
```bash
   python main.py
```
5. Or press the ▶️ Run button in VS Code
